How it works

BCI Technology

Brain-Computer Interface (BCI) technology enables direct communication between the brain and external devices.

MindBand uses advanced neural sensors to analyse signals from the wrist, detecting subtle nerve activity and blood flow patterns linked to cognitive processes. This brainwave activity is recorded as a brainwave pattern.

This pattern is then compared against previously recorded patterns in the watch’s database. If a match is found, the corresponding command is triggered (e.g. "Open notifications").

Custom map commands

Before using MindBand, ensure your smartwatch is paired with your smart home devices, such as lights, thermostats, and speakers.

During setup, you can choose from a list of custom commands. These commands can either control the watch itself, like reading out your last message, or manage smart home devices, such as adjusting the thermostat temperature.

The user is guided to think about the custom command, like turning off a light in the living room. MindBand records the brain’s pattern and maps it with the associated thought. These patterns are then recorded, waiting to be matched in real-time whenever the user thinks about the same action again. Once a match is found, the corresponding command is executed instantly.

Think of something!

Once mapped, simply thinking about the trained action will trigger it in real-time. The system continuously refines and adapts to improve accuracy. Over time, additional actions can be mapped, enabling a more personalised experience.
